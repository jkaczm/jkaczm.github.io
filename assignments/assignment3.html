Assignment 3

For assignment 3, you will only need to write one program. This one program, however, will include multiple functions.

The idea is to write a program in which the user guesses the result of the flip of a coin five times. The program then prints the resulting letter grade based on the percentage of correct predictions.

To create this program, two functions must be defined: 

  1. int coinFlip()
    takes no parameters and returns 1 for heads and 0 for tails.

    the function should return 1 and 0 randomly. See dice.cpp

  2. char letterGrade(double percentage)
    takes as a parameter a double for the percentage of correct guesses and returns the corresponding letter grade based on the following chart:
    Percentage Letter Grade
    100 A
    80 B
    60 C
    40 D
    0 or 20 F

    See Assignment 2

Sample run:


                $ Heads or Tails? 1
                $ The coin landed on heads, correct!
                $ Heads or Tails? 0
                $ The coin landed on heads, incorrect!  
                $ Heads or Tails? 0
                $ The coin landed on tails, correct!
                $ Heads or Tails? 1
                $ The coin landed on tails, incorrect!
                $ Heads or Tails? 1
                $ The coin landed on heads, correct!
                $ You got 60% correct, that's a D!

If you notice, the user enters the integer 1 for heads and 0 for tails when making the prediction.

HINT Here is a general outline for how the code in main should be written:

  • Declare variables to store the input from the user, the number of correct guesses, the number of incorrect guesses.
  • Use a for loop.
  • For each iteration of the for loop:
    • Receive the input from the user
    • Call the function coinFlip
    • Compare the user's input to the result of coinFlip. If they are equal, increment the number of correct guesses by 1. If they are not equal increment the number of incorrect guesses by 1. (Note, those variables should be initialized to 0.)
  • Once the for loop is finished, calculate the percentage of guesses the user got correct, call the function letterGrade and output the result.
