Assignment 2

  1. Write a complete C++ program that does the following

    1. Ask the user to enter a 2-digit number and store it as an int.
    2. If the input is not a 2-digit number, repeatedly ask the same question until the user enters a 2-digit number.
    3. Print all positive factors of the number, in ascending order, one per line. A number i is a factor of n if when you divide n by i, the remainder is 0.
    4. Then print the total number of factors.

    Sample Run

    
                    Enter a 2-digit number: 15
                    Factors:
                    1
                    3
                    5
                    15
                    15 has 4 factors.

    Sample Run

    
                    Enter a 2-digit number: 123
                    Enter a 2-digit number: 7
                    Enter a 2-digit number: 12
                    Factors:
                    1
                    2
                    3
                    4
                    6
                    12
                    12 has 6 factors.
  2. Write a complete C++ program in which the user enters a numeric grade between 0 and 100, and the program prints to the screen the corresponding letter grade according to the following rubric
    Numeric Grade Letter Grade
    90-100 A
    80-89 B
    70-79 C
    60-69 D
    0-59 F

    If the user enters a numeric grade that is not between 0 and 100 inclusive, the program repeatedly asks the user to enter a numeric grade until a proper grade is entered.
