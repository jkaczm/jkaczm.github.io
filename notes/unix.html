Unix Commands

Venus runs the operating system Unix.

Below are some basic Unix Commands for navigating the directory structure, creating and deleting directories, creating and deleting files, as well as a few others.

A Directory is just another term for a folder in Windows or Mac. It is a way to group files and other directories in a heirarchichal structure.

At any given time you are in some directory.

When you fist log on to Venus, you will be located at the root directory or home directory, which is the directory at the highest level in the heirarchy

The directory you are currently in is called the current directory.

The directory that contains your current directory is called the parent directory.



Command Description
ls Lists all the files and folders in the current directory
cd directoryName Changes the current directory to the directory specified by directoryName
cd ~ Changes the curent directory to the home or root directory
mkdir directoryName Creates a directory with the name directoryName
rmdir directoryName Deletes the directory with the name directoryName.
Note, you cannot delete a directory that is not empty.
pwd Prints the full pathname of the current directory
touch fileName Creates an empty file with the name fileName
rm fileName Deletes the file with the name fileName.
Note, this is different than rmdir because this command is only used to delete files, whereas rmdir is only used to delete empty directories
mv oldFileName newFileName moves or renames a file. For example, mv oldFileName newFileName
renames the file file1 to file2.
cp oldFileName newFileName Creates a copy of a file. For example, cp oldFileName newFileName
renames the file file1 to file2.

In addition,

. represents the current directory in which you are located

.. represents the parent directory (the directory above the current directory)

For example, cd .. will change the current directory to the parent directory.



Additional Commands

Command Description
cat fileName Displays the contents of the text file fileName on the terminal.
Can be useful if you just want to see the contents of a file without wanting to edit it in a text editor.
