Problem statement: Write a program that prompts the user enters two integer valued numbers a and b, and prints the result.

Solution:


            #include<iostream>
    
            using namespace std;
    
            int main(){
                int a;
                int b;

                cout << "What is the value of a? ";
                cin >> a;
                cout << "What is the value of b? ";
                cin >> b;

                cout << "The sum of a and b is " << a + b;
                cout << endl;

                return 0;
            }

Note, You can use one cin statement to receive input to the user. For example, the above program can be written as follows.

Alternative Solution:


            #include<iostream>
    
            using namespace std;
    
            int main(){
                int a;
                int b;

                cout << "What are the values of a and b? ";
                cin >> a >> b;

                cout << "The sum of a and b is " << a + b;
                cout << endl;

                return 0;
            }

In the above program, the user can enter the values of a and b on one line with a space separating them. Or the user can type the value of a, hit enter, and then type the value of b and hit enter again.

Experiment on your own to see how exactly input is received from the user.

Problem statement: Write a program that prompts the user to enters his or her name and major, and prints the statement, "Hello, name. Your major is major.". (Note, when you enter data, don't include any spaces, otherwise, only the first word entered will be stored in the variable).

Solution:


            #include<iostream>
    
            using namespace std;
    
            int main(){
                string name;
                string major;

                cout << "What is your name? ";
                cin >> name;
                cout << "What is your major? ";
                cin >> major;

                cout << "Hello, " << name << ". Your major is " << major << "." << endl;

                return 0;
            }

Problem statement: Write a program that prompts the user to enter the lengths of the base and height of a triangle, and prints the area to the screen. (The base and height variables should be of type int)

Solution:


            #include<iostream>
    
            using namespace std;
    
            int main(){
                int base;
                int height;

                cout << "What is the length of the base? ";
                cin >> base;
                cout << "What is the length of the height? ";
                cin >> height;

                cout << "The area of the triangle is " << .5 * base * height << endl;


                return 0;
            }

Problem statement: Write a program that prompts the user to enter his your her height in inches, and prints the user's height in feet and inches to the screen. For example, if the user enters 68. The program will output "5 foot 8 inches".

Solution:


            #include<iostream>
    
            using namespace std;
    
            int main(){
                int height;

                cout << "How tall are you in inches? ";
                cin >> height;

                cout << "You are " << height / 12 << " foot " << height % 12 << " inches" << endl;

                return 0;
            }
