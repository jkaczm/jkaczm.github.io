Venus Account

We will be doing all of our coding in this class on Venus: a server run by the computer science department.

If you are on a Windows computer, the way we connect to Venus is through SSH.

Below is your Venus account username and password

Instructions on how to download and install an SSH client are below.

If you are on a Mac, you don't need to connect to Venus because everything that we do on Venus, you can do on the terminal.

However, if you want to connect to venus through the terminal, you can do so by typing: 


                $ ssh username@venus.cs.qc.cuny.edu

replace username with your username. You will then be prompted to enter your password. To exit the session, type exit

The terminal is located in your Applications/Utilities folder. If this is your first time opening the terminal, you will get a pop-up asking to install Command Line Developer Tools. Click on Install. No need to click on Get Xcode because you do not need it for this class. (However, you should get it for future use).

Username

  • First 2 letters of last name followed by
  • First 2 letters of first name followed by
  • Last 4 digits of CUNYFirst ID

Password

  • 8 digits of CUNYFirst ID

Example

My username is kaja4200. My password is my 8 digit CUNYFirst ID

You can use this account in two places:

  • Computers in the computer lab
  • Venus server (venus.cs.qc.cuny.edu)

Setup Instructions

Windows

Login onto Venus

  1. Download and install SSH
  2. Click Quick Connect
  3. In the field Host Name enter venus.cs.qc.cuny.edu
  4. In the field User Name enter your username as described above
  5. Click Connect. You will then be prompted to enter your password, which is your CUNYFirst ID
