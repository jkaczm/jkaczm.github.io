Text Editors

To edit a file we need to use a text editor. Notepad is an example of a text editor on a Windows machine.

GUI Based Text Editor

There are other GUI based text editors. Some well known free ones are:

  • Sublime Text
  • Visual Studio Code
  • Atom
  • Textmate

Terminal Based Text Editor

The Unix operating system comes with default non GUI text editors such as Vi, Pico, Nano, or Emacs. When a GUI based text editor is not available in your system, you can always use one of these editors. This section is dedicated to Pico/Nano and Vi.

Pico/Nano (EASY)

  • Command to launch Pico/Nano: pico or nano
  • To open a file called my_source_code.cpp: nano my_source_code.cpp
  • To save file: Ctrl+o
  • to quit: Ctrl+x

(note: ctrl+o means you hold down the control key and press o)

Vi (HARD)

This is how to use Vi at the most basic level. Vi is not intended to be used like this. To use Vi correctly, you'll need to invest way more time due to the very high learning curve.

  • Open a file my_source_code.cpp in Vi: vi my_source_code.cpp
  • Vi has 2 modes:
    • Visual Mode (for navigation)

      By default, when you open Vi, you are in Visual Mode.

      In Visual Mode, you cannot type your text. However, you can navigate using your regular arrow keys on the keyboard. Once you are in the position where you want to insert text, press "i" to go to Insert Mode.

    • Insert Mode (for editing)

      In insert Mode, you can type to enter your text. Once you finish entering your text, press "Esc" key to go back into Visual Mode.

  • Save a file in Vi: In Visual Mode, press :w
  • Quit Vi: In Visual Mode, press :q

Again, this workflow above is not how you use Vi. If you have more time and are willing to be proficient in Vi, this is where you can start. Learning to use Vi the proper way is a rewarding investment.

Edit, Compile and Execute Code

Edit

Create an empty file called hello.cpp by typing


        $ touch hello.cpp

Open hello.cpp using pico by typing


        $ pico hello.cpp

Type this in


        #include<iostream>

        using namespace std;

        int main(){
            cout << "Hello, world";
            cout << endl;
            return 0;
        }

Save the file by pressing Ctrl+o

Quit by pressing Ctrl+x

Compile

To compile, execute this command


        $ g++ hello.cpp

Check if a file named a.out had been created by executing the command ls

Execute

Run your newly compiled program by typing


        $ ./a.out

If it is successful, it will print "Hello, World!"

Output Statements


        cout << "Hello";

When this C++ statement is executed, whatever is to the right of << is printed to the console (terminal).

You can use chaining to print multiple literals and variables with a single statement. For example, 


        cout << "Hello."  << "  My name is James";

Prints "Hello. My name is James" to the console on one line

The above statement does the same thing as the following


        cout << "Hello.";  
        cout << "  My name is James";
