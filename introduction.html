Venus

Venus is the name of one of Queens College server where you can upload your files. Since this is an introduction class, we will access and use Venus only on the first few weeks. However, Venus server will be used extensively in higher level classes, make sure you know how to access and use it.

Setup Instructions

Windows

Login onto Venus

  1. Download and install SSH
  2. Click Quick Connect
  3. In the field Host Name enter venus.cs.qc.cuny.edu
  4. In the field User Name enter your username as described above
  5. Click Connect. You will then be prompted to enter your password, which is your CUNYFirst ID

On a Mac, you can use the same Unix commands locally. You don't need to connect to Venus. The terminal is where you can enter these commands. The terminal is located in your Applications/Utilities folder. If this is your first time opening the terminal, you will get a pop-up asking to install Command Line Developer Tools. Click on Install. No need to click on Get Xcode because you do not need it for this class.

However, if you want to access Venus from your Mac, enter the command ssh username@venus.cs.qc.cuny.edu on the terminal, where username is the username as described above. You will then be prompted to enter your password.

Unix Commands

First, some definitions

Directory - If you are familiar with the concept of a folder in Windows or Mac, this is just another term for folder.

When you fist log on to Venus, you will be located at the root directory, which is the folder at the highest level in the heirarchy

Navigating the directory structure

ls lists all the files and folders in your current directory

cd ______ changes the current directory to the directory specified by ______

cd ~ changes directory to the home or root directory

mkdir ______ creates a directory with the name ______

rmdir ______ removes the directory with the name ______

pwd prints the full pathname of the current directory

touch ______ creates an empty file with the name ______

mv ______ ______ moves or renames a file. For example, mv oldFile newFile renames oldFile newFile.

cp ______ ______ creates a copy of a file. For example, cp file1.cpp file2.cpp creates a copy of file1.cpp called file2.cpp

. represents the current directory in which you are located

.. represents the parent directory (the directory above the current directory)
